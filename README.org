* Description

** Généralités

*ProxiBanqueSI v4* est une application offrant une API basée sur les
webservices RESTful. Les différents flux REST (JSON) sont destinés à
être utilisés par une application client destinée aux conseillers
ProxiBanque.

Le projet ProxiBanqueSI fournit une implémentation de référence d'une
telle application client.
